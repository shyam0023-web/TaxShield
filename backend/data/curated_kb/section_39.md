# Section 39 — Furnishing of Returns (GSTR-3B)

## Summary
Section 39 mandates monthly/quarterly filing of returns by every registered person. GSTR-3B is the primary return under this section.

## Key Provisions
- **Section 39(1):** Every registered person (other than composition, ISD, non-resident, TDS/TCS) shall file monthly return in GSTR-3B by 20th of the following month.
- **Section 39(2):** Composition dealers file quarterly return in GSTR-4.
- **Section 39(7):** Return is not deemed filed unless tax is fully paid (self-assessed tax).
- **Section 39(9):** Rectification of errors — omissions in return can be rectified in the return for the month of September/November of the following year or the date of filing annual return, whichever is earlier.
- **Section 39(10):** Late fee for delayed filing — Rs. 50 per day (Rs. 20 for nil return) subject to cap.

## Key Ruling
In **Bharti Airtel Ltd v. Union of India**, the Supreme Court held that GSTR-3B is a return under Section 39 and not merely a summary. Errors in GSTR-3B must be corrected through revision in subsequent returns, not through manual intervention.

## Key Defense Points
1. Technical glitches on GST portal preventing filing are a valid defense against late fee
2. GSTR-3B figures can be corrected in subsequent period returns — no need for amendment
3. Non-filing for 6 months leads to cancellation under Section 29 — file before that
4. Late fee cap has been reduced multiple times by GST Council — check latest notification
5. Interest under Section 50 is payable on net cash liability, not gross liability (per amendment w.e.f. 01.09.2020)
