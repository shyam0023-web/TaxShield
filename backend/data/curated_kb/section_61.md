# Section 61 — Scrutiny of Returns

## Key Ruling
Section 61 empowers the proper officer to **scrutinize** any return filed under the CGST Act to verify its correctness. This is NOT a demand proceeding — it is a pre-demand investigation.

## Scrutiny Process
1. Proper officer identifies discrepancies in GSTR-1, GSTR-3B, or GSTR-9
2. Issues **Form ASMT-10** (notice of scrutiny) specifying discrepancies
3. Taxpayer must respond within **30 days** (extendable up to 30 more days)
4. Response filed in **Form ASMT-11** [See Rule 99(2)]
5. If explanation accepted → dropped, no further action
6. If not accepted → may proceed to determination under Section 73 or 74

## Reply Format: Form GST ASMT-11

### Structure
| Field | Description |
|-------|-------------|
| 1. GSTIN | Taxpayer's GST Identification Number |
| 2. Name | Legal name of the taxpayer |
| 3. Detail of notice | Reference No. and Date of ASMT-10 |
| 4. Tax Period | Period covered (e.g., "SEP-2017 NOV-2017") |
| 5. Reply to discrepancies | Point-by-point response to each defect |
| 6. Amount Admitted & Paid | Table: Sr.No, Tax Period (From/To), Act, Tax(₹), Interest(₹), Others(₹), Total(₹). Use "-NA-" if nothing admitted |
| 7. Verification | Solemn declaration of truthfulness |
| 8. Signature | Authorized Signatory — Name, Designation/Status, Date |

### How to Draft Field 5 (Reply to Discrepancies)
- Address **each discrepancy separately** as listed in ASMT-10
- For each discrepancy, provide:
  - Facts: What actually happened (with dates and amounts)
  - Evidence: Reference DRC-03 challans, GSTR amendments, invoices
  - Reconciliation: Show how the numbers actually match when properly reconciled
- If discrepancy is valid and tax already paid → state payment details with challan reference
- If discrepancy is due to timing/period mismatch → explain with period-wise breakdown
- If discrepancy is incorrect → provide contra evidence with documents

### Amount Admitted Table (Field 6)
If taxpayer has already paid any amount, fill the table:
```
Sr.No | Tax Period (From-To) | Act    | Tax(₹)    | Interest(₹) | Others(₹) | Total(₹)
1     | Apr 2019 - Mar 2020  | CGST   | 20,847.00 | 10,248.00   | -         | 31,095.00
2     | Apr 2019 - Mar 2020  | SGST   | 20,847.00 | 10,248.00   | -         | 31,095.00
```
If nothing admitted, enter "-NA-"

## Key Defense Points
1. Scrutiny is not a demand — no tax liability arises from ASMT-10 alone
2. Respond to EACH specific discrepancy with supporting documents
3. If officer proceeds to Section 73/74 directly without completing scrutiny → **procedural defect**
4. ASMT-10 must specify the exact discrepancy — vague or general notices are challengeable
5. Request extension of time if 30 days insufficient to gather documents
6. Partial payment via DRC-03 before reply reduces the outstanding demand

## Typical Discrepancies Flagged
- GSTR-1 vs GSTR-3B mismatch (outward supplies)
- GSTR-2A/2B vs GSTR-3B mismatch (ITC claimed vs available)
- TDS (GSTR-2A) vs GSTR-3B turnover mismatch (contract receipts)
- Late filing of returns
- Non-filing of annual return GSTR-9

## Escalation Path
If ASMT-11 reply is not accepted:
1. Officer issues **Show Cause Notice** under Section 73(1) or 74(1)
2. Taxpayer gets 30 days to reply to SCN
3. Section 73(8): Pay within 30 days → no penalty
4. If not paid → adjudication order under Section 73(9) with penalty
